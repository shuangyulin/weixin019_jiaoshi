随着我国经济迅速发展，人们对手机的需求越来越大，各种手机软件也都在被广泛应用，但是对于手机进行数据信息管理，对于手机的各种软件也是备受用户的喜爱，微信被用户普遍使用，为方便用户能够可以随时教师管理系统信息管理，特开发了基于微信小程序教师管理系统。
